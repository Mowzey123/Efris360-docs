EFRIS360

Getting Started

After system initialization and setup, you will be asked to provide email addresses for your users that will be programatically accesing the API. Upon succesful intiailization a secrete key will be sent to the respective email address, please note that this is highly sensitive peiece of data and should be securly kept.

Endpoint: The end-point to send data to will alsobe shared by email after successul setup of the offline enabler.

Authorization

To acquire a valid jason web token send data to authorization interface

Interface : /efris_v.0.0.1/authManagement

Method : /generateAuthToken

Sample Request Body
{
    "tinno":"1014650988",
    "email":"example@gmail.com",
    "seckey":"{{seckey}}"
}
Key Description
tinno Your tin number
seckey Acquired secret key

Renew Authorization Token

The issued JWT above expires within 2hrs period, to renew the token before expiration perfom the request descirbed below.

Interface : /efris_v.0.0.1/authManagement

Method : /renewAuthToken

Sample Header Body
{
    "Authorization":Bearer {{token}}
}
Sample Request Body
{
    "seckey":"{{seckey}}"
}
Item Description
token JWT acquired on succesful authorization
seckey Your acquired secret key

            "goodsCode": "001",
            "measureUnit": "per stick",
            "unitPrice": "6999.99",

            "haveExciseTax": "yes",
            "description": "apple test",
            "stockPrewarning": "10",
            "havePieceUnit":"yes"
        }
    ],
    "seckey":"{{seckey}}"
}
Key Description
goodsName Name of the item. (string)
goodsCode Unique item ID known to you. (string)
measureUnit Item measure unit. Enum: ["per stick","per stick"]
unitPrice Item unit price. (string)
measureUnit Item measure unit. Enum: ["per stick","per stick"]
commodityCategoryId Level 4 Categpory ID. (string)
haveExciseTax Item measure unit. Enum: ["yes","no"]
description Item description. (string)
stockPrewarning Stock prewarning levels. (string)
havePieceUnit If item has peice unit. Enum: ["yes","no"]
seckey Your acquired secret key
